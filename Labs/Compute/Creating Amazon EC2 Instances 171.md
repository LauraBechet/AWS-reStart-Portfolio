<h2>Overview</h2>

<p>
AWS provides multiple ways to launch an Amazon Elastic Compute Cloud (EC2) instance.
</p>

<p>
In this lab, the AWS Management Console is used to launch an EC2 instance that acts as a <strong>bastion host</strong>. 
From this bastion host, another EC2 instance is launched which will run as a <strong>web server</strong>.
</p>

<p>
EC2 Instance Connect is used to securely connect to the bastion host, and the 
<strong>AWS Command Line Interface (AWS CLI)</strong> is used to launch the web server instance.
</p>
