<h2>Introduction to Amazon EC2</h2>

<h3>Overview</h3>
<p>
Amazon Elastic Compute Cloud (EC2) is a cloud service from AWS that lets you create and run virtual servers on the internet. 
It allows you to quickly get computing power without buying physical hardware. 
You can start, stop, or resize servers whenever you need them, making it easy to build and scale applications.
</p>
