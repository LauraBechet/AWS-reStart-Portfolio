<h2>Introduction to Amazon EC2</h2>

<h3>Overview</h3>
<p>
Amazon Elastic Compute Cloud (EC2) is a cloud service from AWS that lets you create and run virtual servers on the internet. 
It allows you to quickly get computing power without buying physical hardware. 
You can start, stop, or resize servers whenever you need them, making it easy to build and scale applications.
</p>

<h3>Topics Covered</h3>

<ul>
  <li>Launch an EC2 instance with <strong>termination protection</strong> enabled</li>
  <li><strong>Monitor</strong> your EC2 instance performance</li>
  <li><strong>Edit the security group</strong> to allow HTTP (web) access</li>
  <li><strong>Resize the instance</strong> to increase or decrease computing power</li>
  <li><strong>Test termination protection</strong> to prevent accidental deletion</li>
  <li><strong>Terminate the EC2 instance</strong> when it is no longer needed</li>
</ul>
